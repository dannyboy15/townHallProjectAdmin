Location Errors

These had problems with the geocoding.
To update:
  • Enter a new address and click "Geocode!"
  • Make sure the address is correct in the preview
  • Click "Update" to save changes to the database
If there is no actual address, leave the street address blank.
If there is something like a room number, that might be the problem, maybe put that in the "location" field instead.

All current public events

These are the events currently on the public site.
If an event has occurred, click "archive" to remove from the public site (will still be saved in the database).
If an event has been canceled, click "delete" to remove it from the database all together.

Date errors

These had problems with assiging a date, mostly because they are repeating events.
To update:
  • enter a new Date or Time
  • Make sure the date and time is correct in the preview
  • Click "Update" to save changes to the database
If the date has a '22nd' please change it to 22.
If the date looks fine, it may be the time or time zone.
